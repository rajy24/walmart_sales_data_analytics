# Walmart Sales Data Analytics Project

![Banner](walmart_img.png)

## Project Overview

This project analyzes Walmart sales data using Python, PostgreSQL, and SQL.

The dataset was obtained from Kaggle, cleaned using Pandas, and loaded into PostgreSQL for analysis.

The project focuses on solving 10 real-world business problems using SQL queries and analytical techniques.

## Business Problems Solved

1. Find different payment methods, quantity sold and transactions.
2. Identify highest rated category in each branch.
3. Identify busiest day in each branch.
4. Calculate quantity sold per payment method.
5. Determine avg, max and min ratings by city.
6. Analyze ratings for Little Elm city.
7. Calculate category-wise profit.
8. Find preferred payment method per branch.
9. Categorize sales into Morning, Afternoon and Evening shifts.
10. Identify branch with highest revenue decrease ratio.
